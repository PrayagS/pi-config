# pi-footer

Custom Pi footer with context gauge and provider usage bars.

Auto-detects current provider from selected model and renders provider-specific usage data in footer. Forked from `@ogulcancelik/pi-minimal-footer`.

## Features

- context usage display
- provider quota / rate-window bars
- extension status row filtering and ordering
- fish-style cwd abbreviation in footer
- auth lookup from env, `auth.json`, and some provider-specific fallbacks
